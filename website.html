Portfolio Website Case Study

Project Overview

Understanding front-end development is a fundamental aspect of web design and development. By creating and developing a portfolio website, I can effectively display my proficiency in HTML, CSS, and JavaScript. This platform not only highlights my technical abilities but also serves as a showcase for the diverse UX/UI design projects I have worked on. Through this portfolio, I aim to present a comprehensive view of my skills and experience, demonstrating my capacity to blend aesthetics with functionality in web design.

Role

  • UI/UX Designer
  • Front-end Developer

Duration

  • 5 weeks

Tools

Problem

As a UI/UX designer, it's essential to highlight my expertise in web design and development to attract potential clients and employers. Developing a portfolio website will provide the perfect platform to showcase my skills and knowledge, while also offering opportunities to network with other professionals in the industry.

Solution

Gaining knowledge in front-end languages like HTML, CSS, and JavaScript will equip me with the tools to build a portfolio website. The site will be simple and user-friendly, while also showcasing my design expertise.

Process

Research

As part of the project, I consulted various portfolio websites and many were very creative and showcased the designers creativity and knowledge. However, early on I kew that I wanted a clean and simple look, so anyone could access quickly certain kee information, like a resume and past projects information.

Define

Following my research, I had to identify the essential elements and features to include in the portfolio website. The site will feature the following sections:

  • Home
  • Project Pages
  • About me
  • Contact

Styling Guide

Color

The reason I chose these colours is because they show harmony, simplicity and seriousness at the same time. I also think this particular red is perfect to be used on buttons and various actions.

Typography

Poppins

Aa Bb Cc Dd Ee Ff Gg Hh Ii
Jj Kk Ll Mm Nn Oo Pp Qq Rr
Ss Tt Uu Vv Ww Xx Yy Zz

I chose the "Poppins" font because it is a font that is quite simple, clean and easy to read in the eye of the viewer. It also has several variations so I can play with them in headings, subheadings and body text.

Develop

HTML was the language I started coding with using Visual Code Studio to build the website as a whole. The basic layout was set up so that the images and texts (as headers, paragraphs, etc.) could be added to it. This means that throughout the development process, I had to make sure that my code is clean enough for easier reading or else editing will become impossible. After setting up HTML for the basic structure, I started applying CSS styles to my website. This moment gives you a chance to envision your web page. It's really fun designing buttons, headings, links and pictures. Then, I designed pages layout starting with mobile first strategy to ensure that it will look good on any devices. Lastly, a responsive hamburger menu was created by JavaScript for mobiles.

User Testing

  • Task 1: You've opened up the Portfolio Site. The page opens. You would like to learn more about the project called Creo Apparel. Where do you think you can find more information about that project? What was the project about?
  • Task 2: You've browsed through all his projects and become more interested in his former education and experiences. Now, you want to see his resume. Where do you think you can find it?
  • Task 3: You've downloaded the resume and are quite impressed that he learned all his skills in only the last couple of months. For you, this means that he's a quick learner and highly motivated. You want to reach out to him to share the current job opening with him. How might you proceed?
